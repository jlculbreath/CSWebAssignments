How to Create a GitHub Repository

First, a brief description.

On this page, I will explain how easy it is to create a remote GitHub repository.

  1. Go to the github.com website
  2. If you do not have an account, create one.
    We will assume you already have an account.
  3. Click on the link to create a new repository
  4. Fill in the title, description, and any other info you like.
    We recommend always adding a readme file.
  5. Click create repository
It may take a few moments to finish, but that's it!
